Обзор проекта:
    Просто todo-list сделано на React

Инструкции по запуску проекта локально,
    'git clone https://github.com/Dumba21/toDO-list.git'
    cd todo-list
    npm start

    первый input для поиска задач, рядом кнопка фильтрации, внизу кнопка "+" добавления
